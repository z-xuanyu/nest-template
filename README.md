# nest-template

nestjs 通用基础模板，server 分为 admin 端和 web 端

## Node 版本要求

```javascript

需要 Node.js 10.0 或更高版本 (推荐 12.0+)

```

## 项目运行

```javascript
git clone git@github.com:z-xuanyu/nest-template.git

cd nest-template/server

yarn install

//admin 端
nest start -w admin

// web 端
nest start -w web

```
